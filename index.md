### Lung Cancer Detection

[![Model Architecture](https://img.shields.io/badge/Model_Architecture-orange)](https://github.com/SSDivyaRavali/Image-Classification/blob/temp_branch/docs/VGGmodel.md) [![View on GitHub](https://img.shields.io/badge/GitHub-View_on_GitHub-blue?logo=GitHub)](https://github.com/SSDivyaRavali/body-parts-recognition)

<div style="text-align: justify">The goal of this project is to detect lung cancer from CT scan images.<a class ="hlink" href="https://www.kaggle.com/datasets/mohamedhanyyy/chest-ctscan-images">Here</a> is the dataset analyzed in this project </div>

- Built an end-to-end ML pipeline for lung disease classification using CT scan images, covering data ingestion, training,
evaluation, and deployment.Implemented experiment tracking and reproducibility using MLflow, DagsHub, and DVC for data
and model versioning.
- Fine-tuned a VGG16 CNN with frozen convolutional layers and custom fully connected layers on a CT image dataset.Deployed
the model via a Flask web application enabling users to upload CT images for real-time predictions.
- **Tech Stack:** Flask, Tensorflow, DVC, MLFlow, GitHub, DagsHub. Models: VGGNet-16
